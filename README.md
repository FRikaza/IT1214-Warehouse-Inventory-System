# IT1214 - Warehouse Inventory System

## Project Description
This is a Java-based console application designed to manage warehouse stock. It allows users to register new products with unique IDs, track quantities, and monitor pricing. It serves as a fundamental backend logic example for inventory management.

## How to Run the Program
1. Ensure you have **Java JDK** installed on your system.
2. Download the `Main.java` file.
3. Open your terminal or command prompt.
4. Compile the code:  
   `javac Main.java`
5. Run the program:  
   `java Main`
